Canvas Divider(cDiv)

This extension aims to provide a simple way to divide the canvas (or a selection) into equal sections.

Basic Usage

To begin select cDiv in the Tools tab. If you have a document open, A window should pop up where you can specify your how many divisions to make, as well as whether or not to keep any guides you may already have on the page.

  • The Horizontal Guides option specifies how many horizontal divisions should be made.
  • The Vertical Guides option specifies how many vertical divisions should be made.
  • The Preserve Guides option specifies whether or not to remove pre-existing guides.
  • The Border Guides option specifies whether or not to add guides to the borders of your selection. You can add a selection to just the borders by checking this and creating 0 divisions.

Dividing a Selection

If you create a selection before open the dialog, the selection will be divided instead of the entire canvas.

Installation

To install cDiv, navigate to the releases category and download the most recent release, which should be a zip file. Unpack this zip file into the pyKrita folder in your resources folder. If you are unsure where your resources folder is, open krita and check in settings > configure Krita > general > resources
